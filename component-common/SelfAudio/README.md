## 自建音频播放

## api
| 属性                    | 说明                       | 类型 | 默认值 
|:----------------------------|:----------------------------------|:-------------------------|:------
| source | 必选，音频源 | string | -
| poster | 可选，音频海报 | string | -
| name | 可选，音频名称 | string | -
| frozen | 是否禁止滑动 | bool | false (默认可滑动)
| jump | 可否跳转 | bool | true (默认可跳转)
| autoPlay | 自动播放 | bool | false
